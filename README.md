# Agentic AI Workflow System

Agentic AI Workflow System is an interactive Streamlit-based platform that allows users to design and evaluate AI agent workflows using scenario-based tasks.

Users receive real-world scenarios such as travel booking or appointment management and arrange AI agents in the correct workflow sequence. The system validates workflow logic, orchestration patterns, missing agents, and approval dependencies using a rule-based AI evaluation engine.

## Features

* Multiple AI workflow scenarios
* Workflow builder system
* AI agent orchestration validation
* Workflow scoring engine
* Feedback and correction system
* Streamlit interactive UI
* Deployable cloud-ready application

## Tech Stack

* Python
* Streamlit
* Pandas

## Run Locally

```bash
pip install streamlit pandas
streamlit run app.py
```

## Deployment

The project is deployed using Streamlit Cloud.
